##  About

**Video Downloader Helper** is a powerful Chrome extension that helps you download videos from any website.

##  Features

- **Video Detection**
- **Multiple Formats**
- **Batch Download**
- **Quality Selection**


##  Installation

1. Download or clone this repository
2. Open Chrome  `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked**  Select folder

##  Privacy

-  No data collection
-  No analytics
-  Open source
