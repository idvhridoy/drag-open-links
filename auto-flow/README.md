##  About

**Auto Flow** is a powerful Chrome extension that automates video generation with Google's VEO AI. Queue multiple prompts and let the extension handle the generation process automatically.

##  Features

- ** Prompt Queue** - Add multiple prompts and process them automatically
- ** Configurable Settings** - Adjust delay, model, and other parameters
- ** History Tracking** - View past generations and their status
- ** Keyboard Shortcuts** - Quick controls for power users
- ** Auto-Retry** - Automatically retry failed generations
- ** Auto-Download** - Download generated videos automatically

##  Installation

1. Download or clone this repository
2. Open Chrome  `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked**  Select folder

##  Usage

1. Navigate to VEO AI website
2. Click extension icon
3. Add prompts to queue
4. Click Start to begin automation
