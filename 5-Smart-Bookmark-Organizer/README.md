##  About

**Smart Bookmark Organizer** is a powerful Chrome extension that helps you organize bookmarks intelligently.

##  Features

- **Auto Categorization**
- **Search**
- **Tags**
- **Export/Import**


##  Installation

1. Download or clone this repository
2. Open Chrome  `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked**  Select folder

##  Privacy

-  No data collection
-  No analytics
-  Open source
