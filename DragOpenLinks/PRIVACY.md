# Privacy Policy for Drag Open Links

**Last Updated: November 2024**

## Overview

Drag Open Links ("the Extension") is committed to protecting your privacy. This privacy policy explains how the Extension handles your data.

## Data Collection

**We do not collect any data.** The Extension:

- ❌ Does NOT collect personal information
- ❌ Does NOT track your browsing history
- ❌ Does NOT send any data to external servers
- ❌ Does NOT use analytics or tracking tools
- ❌ Does NOT store any information about the links you open

## Data Storage

The Extension stores only your preferences (settings) locally on your device using Chrome's built-in storage API. This includes:

- Maximum tabs per drag setting
- Open delay setting
- Toast notification preference
- Counter badge preference

This data:
- Is stored locally on your device only
- Is never transmitted anywhere
- Can be cleared by removing the extension

## Permissions

The Extension requires the following permissions:

### `tabs`
- **Purpose**: To open new tabs when you select links
- **Usage**: Only used when you actively drag and release to open links
- **Data Access**: No browsing history is accessed or stored

### `storage`
- **Purpose**: To save your preferences locally
- **Usage**: Stores only the settings you configure in the popup
- **Data Access**: Only your extension settings, nothing else

### `<all_urls>` (Host Permissions)
- **Purpose**: To allow the extension to work on any webpage
- **Usage**: The content script needs to run on pages to enable drag selection
- **Data Access**: No page content is collected or transmitted

## Third-Party Services

The Extension does not use any third-party services, analytics, or tracking tools.

## External Links

The Extension popup contains links to:
- GitHub repository (for source code and issues)
- Buy Me a Coffee (for voluntary donations)

Clicking these links will open external websites subject to their own privacy policies.

## Data Security

Since no personal data is collected or transmitted, there is no risk of data breach from using this Extension.

## Children's Privacy

The Extension does not collect any information from anyone, including children under 13 years of age.

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date at the top of this document.
