## Justification for Permissions

### tabs
"Required to create new browser tabs when the user selects and releases links. The extension only creates tabs when explicitly triggered by the user's drag action."

### storage
"Required to save user preferences locally on the device. This includes settings like maximum tabs, delay time, and UI preferences. No data is transmitted externally."

### host_permissions (<all_urls>)
"Required to inject the content script that enables drag selection functionality. The script only handles mouse events for link selection and does not access or transmit any page content."
